# Answers

```python
from quran_foundation import QuranClient

client = QuranClient(client_id="YOUR_CLIENT_ID", access_token="APP_ACCESS_TOKEN")

answers = client.get_answers_by_ayah("2:255")
answer = client.get_answer(988)
counts = client.count_answers_within_range("2:255", "2:256")
```

Supported helpers:

- `get_answers_by_ayah(verse_key, params=None)`
- `get_answer(question_id)`
- `count_answers_within_range(from_verse_key, to_verse_key, params=None)`
