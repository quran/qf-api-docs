## 📙 Translation Fields (`translation_fields`) {#translation-fields}

These fields can be used in the `translation_fields` query parameter:

| Field | Description |
|-------|-------------|
| `chapter_id` | Chapter number of this verse |
| `verse_number` | Verse number within the chapter |
| `verse_key` | Verse key (e.g., 2:255) |
| `juz_number` | Juz number this verse belongs to |
| `hizb_number` | Hizb number this verse belongs to |
| `rub_el_hizb_number` | Rub el Hizb number for the verse |
| `page_number` | Mushaf page number |
| `ruku_number` | Ruku number this verse belongs to |
| `manzil_number` | Manzil number this verse belongs to |
| `resource_name` | Name of the translation resource |
| `language_name` | Name of the translation language |
| `language_id` | Language ID |
| `id` | ID of the translation |

## 📕 Tafsir Fields (`tafsir_fields`) {#tafsir-fields}

These fields can be used in the `tafsir_fields` query parameter:

| Field | Description |
|-------|-------------|
| `chapter_id` | Chapter number of this verse |
| `verse_number` | Verse number within the chapter |
| `verse_key` | Verse key (e.g., 2:255) |
| `juz_number` | Juz number this verse belongs to |
| `hizb_number` | Hizb number this verse belongs to |
| `rub_el_hizb_number` | Rub el Hizb number for the verse |
| `page_number` | Mushaf page number |
| `group_verse_key_from` | Grouped tafsir start verse key |
| `group_verse_key_to` | Grouped tafsir end verse key |
| `group_tafsir_id` | Group tafsir ID |
| `start_verse_id` | Start verse ID |
| `end_verse_id` | End verse ID |
| `resource_name` | Name of the tafsir resource |
| `language_name` | Name of the tafsir language |
| `language_id` | Language ID |
| `id` | ID of the tafsir |
